# 采样理论（Jazz-hop oriented）

> 目标：建立采样的系统理论框架（工作流 + 6维度），并形成可复用的 jazz-hop 制作流程与判断标准。

---

## 第1节｜三种工作流（loop / chop / resample）

### 1) Loop-based（整段循环）
取 **1–8 小节**循环当作主底。

**优势**
- 最快构建出骨架

**劣势**
- 原创性低，容易单调

**Jazz-hop 里的 loop 处理**
- 滤波
- 空间
- 断点
- 去鼓点 / 补底噪

---

### 2) Chop-based（切片重组）
将素材切成多个 **chops**，用新的节奏将它们重组。

**优势**
- 原创性强
- 可控旋律
- groove 可塑性强

**劣势**
- 费时间
- 节奏审美门槛高

**Jazz-hop 中的 chop**
- 中等碎度：**8–16 chops / 段**
- 问答感 / 留白感 / 旋律碎片化

---

### 3) Resample-based（再采样）
先对采样做一轮「音色塑形」（滤波 / 饱和 / 空间 / 降质 / 压缩等），再把处理后的结果录回音频，把它当作新素材再次切片/编排。  
流程 **不可逆**（需要版本管理）。

**优势**
- 将零散片段固化为统一质感
- 给声音赋予特殊性

**常用路径**
- loop 定氛围 → resample 定质感 → chop 定编排与原创性

---

## 第2节｜采样的 6 个维度（time / pitch / timbre / rhythm / space / arrangement）

### 1) Time（时间 / 速度 / 对齐）
采样与 BPM、鼓的网格关系。

**三种策略**
1. 严格对齐：稳、干净  
2. 半对齐：松、groove  
3. 不对齐：漂浮  

**Jazz-hop 的 time**
- 鼓和采样的重拍必须和谐
- 其它位置允许松，但要像演奏而不是错拍

---

### 2) Pitch（音高 / 调性 / 和声归属）
采样与贝斯 / 人声 / 和弦的和声一致性。

**两种主宰关系**
1. 采样主宰：让 bass / 和弦 去贴采样  
2. 自己主宰：让采样转调去贴自己的调性  

**Jazz-hop 的 pitch**
- bass 的根音是否和采样的和声打架
- 采样转调后有没有出现怪异芯片感（转太多音就容易出现）

---

### 3) Timbre（音色 / 质感 / 唱片感）
采样听起来是原曲还是再创作。

**常用手段**
1. EQ（变暗、去刺、去浑）  
2. 轻饱和 / 磁带感（中频胶感）  
3. 轻压缩（粘合）  
4. 降质 / 噪声（少量纹理）  

**Jazz-hop 的 timbre**
- 鼓不会显得奇怪（整体世界观统一）

---

### 4) Rhythm（节奏：落点 / swing / 偏移）
采样片段怎么动起来。

**核心：落点审美**
1. 哪些 chop 要落在强拍  
2. 哪些要提前/滞后制造呼吸  
3. 留白：给鼓 / 贝斯 / 人声留出空间  

**判断**
- 律动是否成立
- 旋律碎片是否有问答感而不是乱拼

---

### 5) Space（空间：远近 / 深浅 / 宽窄）
氛围与层次。

**常见布局**
1. 主体（钢琴/旋律）偏近：短房间 / 少混响  
2. 背景空气偏深：长一点的 room / plate + 很轻的 delay  
3. 低频更 mono，高频更宽（避免糊）  

**判断**
- 听起来有没有空间的呼吸感
- 鼓进来后空间是否塌掉（塌掉说明空间链不对）

---

### 6) Arrangement（结构：什么时候变、变多少）
耐听与叙事，避免单调循环。

**Jazz-hop 常见结构**
1. A（主 loop）：最完整  
2. A’（变化 loop）：少一个 chop / 更暗 / 更干  
3. B（对比段）：换和声片段或换切片节奏  

**过渡**
- 滤波、断点、填充、短人声 one-shot

**判断**
- 每 4/8/16 小节有没有小变化

---

## 第3节｜滤波（Filter）

滤波是在音频里用一个频率筛子，把某一段频率保留，另一段频率削弱/切掉。  
本质是对声音做频段选择，从而改变音色（明暗/薄厚/远近）。

### 1) 低通 Low-pass（LPF）
让低频通过，高频被削弱。  
听感：声音变暗、更柔和、像隔着门/窗帘。

**参数**
- cutoff（截止频率）：决定从哪开始变暗  
- resonance（共振）：在截止点附近凸起一点（加一点波动的感觉）

### 2) 高通 High-pass（HPF）
让高频通过，低频被削弱。  
听感：更薄、更干净、去掉轰隆和浑浊。  
用途：采样去低频，给 kick / 808 / bass 让空间。

**参数**
- cutoff：开得太高会把温暖感削没

### 3) 带通 Band-pass（BPF）
只放行中间一段频率，低和高都削。  
听感：电话音，聚焦于中频。  
用途：过渡、build、让采样像从远处来。

**Jazz-hop 常见滤波用法**
1. HPF 轻切：去掉采样里多余低频，让 bass 更稳  
2. LPF 轻闷：让采样变暖、不给 hi-hat 和人声抢空间  
3. 自动化滤波：段落过渡时把 LPF 从暗开到亮（打开窗帘）

---

## 第4节｜频率与音高

频率与音高（pitch）强相关，但不是同一概念。

- **音高**：声音的基频高低  
- **频率**：声音中包含很多频率成分，包括基频、泛音、谐波以及各种噪音，共同决定音色  

滤波虽然可能会让音高听起来像变了，但本质上改变的是各频率的音量，而不是改变频率本身。
- 滤波 = 改能量大小  
- 变调 = 改频率位置  

---

## 第5节｜Jazz-hop 的通用制作流程（6维度工作顺序）

先用两个维度快速定基调，再用两个维度提高原创性与律动，最后用两个维度做成品化与耐听结构。

### Step 1：Time + Timbre（先让它成为一张唱片）

#### 1) Time
让采样与工程形成可信的时间关系。  
只追求一件事：鼓进来不会像与伴奏打架。  
首先做到：第一拍与采样的主重音能对上（或有意地稳定偏移）。

**产出物**
- 一个能循环的 4/8 小节采样底子

#### 2) Timbre
把原采样赋予自己的特色。初期只做三件事：
1. 去浑（低频整理）  
2. 变暖（轻饱和/胶感）  
3. 变暗（轻低通/去刺）  

**产出物**
- 区别于原曲、已经“自洽”的采样底色（鼓组还没加入但已经和谐）

---

### Step 2：Pitch + Rhythm

#### 3) Pitch
确定谁（采样 or 自己想要的调）来主宰调性。

**产出物**
- 明确重心音（根音/调性），为后续 bass 和和弦服务

#### 4) Rhythm
用 chop 或触发方式让它产生“问答与留白”。

**产出物**
- 一个循环小节的 groove loop

---

### Step 3：Space + Arrangement（从 loop 变成作品）
此时可以加入基础鼓组，确立稳定的律动框架。

#### 5) Space
做近的主体 + 深的房间。

**产出物**
- 鼓不糊的情况下，仍保持房间感的空间层次

#### 6) Arrangement
结构编排（小变化）。
- 少一个 chop、更暗、更干、fill 填充、幽灵军鼓

**产出物**
- 至少两种 loop 状态（A、A’、B 中至少两种）的作品雏形

---

## 第6节｜Jazz-hop 素材选段

### 三种素材
1. **氛围型**：没有突出旋律/和弦伴奏纹理丰富；结尾有自然房间尾巴残响；容易无缝循环；适合做 loop 底色  
2. **旋律型**：适合做 chop 触发；清晰音头、明显断句；可做问答的小动机  
3. **变化型**：和声突然转了一下或编制变了（突然空/突然加乐器）；适合做 B 段或过渡  

### 选段硬标准
1. 循环耐听  
2. 低频干净（最好不要有满的贝斯、底鼓等低频）  
3. 尾巴与衔接点和谐：在呼吸点做切口形成自然尾音  

### 处理不干净低频的方法
1. 直接用 loop 中的低频  
2. EQ 削弱/砍掉 loop 低频（30–120Hz），再用自己的鼓压过去；同时可压一下 2–8kHz 的 hi-hat 存在感  
3. 音轨提取去掉鼓轨  

### 素材资源
- **samplette.io**：数字挖碟工具；类似随机轮盘从 YouTube 抽取音乐视频，并提供筛选条件让你更快找到适合采样的曲子（需要翻墙）

## 第7节｜Jazz-hop 的和声管理

在采样自带和声信息时，如何避免和声叠加时冲突。

### 基础概念：采样包含三层和声信息
1. 根音：决定和弦地基  
2. 和弦  
3. 旋律  

**本质**
- 让第一层稳定，允许第二层丰富，让第三层自由但可控

**核心原则**
- 低频只能有一个根音作为地基
- 上层旋律可以非常自由

### 三种和声叠加策略
1. **loop 底色主宰**：loop 提供根音和弦；chop 做上层旋律；bass 贴着根音走  
2. **chop 主宰**：chop 当主和声；loop 退化为空气层  
3. **拼贴主宰**：故意保留冲突但保持根音稳定；冲突发生在中高频营造 lo-fi 梦幻感；适合段落过渡  

---

## 第8节｜Groove：Swing 是重音位置的美学
Swing：把原本平均分的细分拍拉成不平均的松紧感，让律动摇摆。  
它是系统性的时间偏移，往往只拖动某一些位置（laid-back / shuffle-ish）。

---

## 第9节｜采样的再语境化（Recontextualization）
再语境化：改写原曲的情绪、叙事、风格。

### 四个框架
1. **节奏**
   - 规整 → 干净  
   - 后靠 swing → 慵懒  
   - 留白 → 孤独  
2. **低音重心（根音）**
   - 同一段和弦，改变根音会带来不同的和声功能：温暖 ↔ 悬疑；孤独 ↔ 团聚  
   - 根音掌控情绪方向  
3. **音色**  
4. **空间**
   - 干、近：像室内、耳语  
   - 湿、深：像回忆、像雨夜  
   - 宽/窄：开放/孤独  

---

## 第10节｜Jazz-hop 像旧照片的回忆：结构性解释（暗、暖、胶、噪、深）

旧照片的典型特征：
- 细节损失（像高频被削）  
- 色彩偏暖（像中频更厚）  
- 有颗粒（像底噪纹理）  
- 有景深（像空间层次）  

Jazz-hop 把这套视觉语言移植到声音里，所以你会觉得“像回忆”。

### 五个关键词
- **暗**：高频被抹掉 → 营造隔着墙的暗感  
- **暖**：能量集中于中频 → 温暖感  
- **胶**：上层旋律自由展开，但低频根音统一 → 自由混乱中蕴含秩序  
- **噪**：底噪/空气音补齐房间与时间（避免无菌数字音）；需克制，否则廉价 lo-fi  
- **深**：混响层次清晰：近景在面前，远景（房间）在身后，中间隔着空气层  

### 三种 Jazz-hop 场景模板
1. **下午室内（comfort）**：暖为主、暗适中、噪极轻、深度适中；温暖、轻松、陪伴  
2. **夜路（noir）**：更暗、更深、更噪、更胶；孤独、电影、雨夜  
3. **梦游回忆（dreamy）**：暧昧深度与模糊边缘；漂浮、迷离、闪回  

---

## 第11节｜A / A’ / B 最小结构与变化规则

### 底层观念
结构 = 信息的重新分配。

耐听结构来自：
- 同样的素材，换信息密度  
- 同样的 loop，换层级分工  
- 同样的和声床，换旋律出现的频率  

### A / A’ / B 三段式
- **A（主叙事）**：底色 + 主鼓 + 旋律  
- **A’（同叙事的小变化）**：不换叙事，只换语气（更空/更暗/旋律更少/鼓更轻），使 A 回来时更有变化感  
- **B（对比段）**：镜头切换；换采样片段或换旋律，制造段落边界  

**样板结构**
- A → A’ → A → B

---

## 第12节｜去相似化的五种方法（降低“像原曲”的风险）

1. **重构**：用 chops 重新组合  
2. **遮蔽**
   - 频段遮蔽：弱化容易被识别的频段  
   - 时间遮蔽：只取一句话的一部分  
   - 背景遮蔽：把它变成背景纹理（远、暗、碎）  
3. **替换**：替换旋律 / 根音 / 鼓  
4. **再语境化**  
5. **resample**
